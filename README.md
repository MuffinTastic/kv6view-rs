# kv6view-rs
A simple KV6 voxel model viewer written in Rust. Something I threw together in a day for fun.

Configuration is hardcoded in `src/controls.rs`.

![caco.kv6](kv6view-rs.png)

## Compiling and running

`cargo run -- <file>`
